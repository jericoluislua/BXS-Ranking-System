# BXS-Ranking-System
A BXS-specific tool for calculating player/blader rankings.

This is the current point calculation and may be updated depending on circumstances and changes to the ranking system.

# General point distribution

participation = 1 points

Swiss round win = 1 points

Swiss round loss = 0

Swiss King = Swiss round wins x 2 = \# points

# Top 8

1st = +15 points

2nd = +12 points

3rd  = +9 points

4th = +6 points

rest = +3 points

# Top 16:

1st = +18 points

2nd = +15 points

3rd = +12 points

4th = +9 points

5th = +6 points

rest = +3 points
